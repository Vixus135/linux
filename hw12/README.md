## Содержание упражнений
### Упражнение 1: Система обработки сообщений (папка `mess`)
**Задача:** Реализовать систему, где один поток генерирует сообщения через
неопределенные промежутки времени, а другой поток их обрабатывает.

### Упражнение 2: Пул потоков с разделенными мьютексами (папка `mutex`)
**Задача:** Реализовать сервер с пулом потоков, где главный поток добавляет
запросы, а несколько потоков-обработчиков их обрабатывают.

Требования для сборки и запуска
-------------------------------
- Компилятор gcc
- Библиотека pthreads
- Операционная система Linux/Unix

Для просмотра каждого упражнения перейдите в соответствующую папку и прочитайте файл README.md 
(в нём описана логика кода и инструкция по компиляции)
