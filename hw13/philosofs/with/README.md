# Упражнение 3.2: Обедающие философы
## Задание
Реализовать задачу об обедающих философах так, чтобы не возникал дедлок:
- За столом сидят 5 философов
- Чтобы поесть, каждому нужны две вилки — левая и правая
- Если философ не может взять обе вилки сразу, он должен положить вилку обратно и ждать

## Как работает программа
- Каждый философ — отдельный поток
- Вилки — это массив мьютексов `forks[]`
- Есть общий мьютекс `global_mtx` и условная переменная `cond`, чтобы философы могли ждать и будить друг друга
- Алгоритм:
  1. Философ думает.
  2. Пытается взять обе вилки с помощью `pthread_mutex_trylock`
  3. Если удалось — ест, потом кладёт вилки и будит остальных (`pthread_cond_broadcast`)
  4. Если не удалось взять обе вилки — отпускает захваченную и ждёт сигнала на условной переменной.
- Такой подход исключает дедлок, потому что философ никогда не держит одну вилку слишком долго

## Файлы проекта
- `main.c` — создаёт потоки философов и инициализирует вилки
- `funcs.c` — содержит функцию `philosopher()`, которая описывает поведение философа
- `Makefile` — автоматическая сборка проекта

## Компиляция и запуск
Собрать программу:
```bash
make
```
Запустить:
```bash
./philo
```
Программа работает бесконечно. Чтобы остановить — нажмите Ctrl+C
