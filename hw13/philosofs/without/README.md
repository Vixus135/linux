# Упражнение 3.1: Обедающие философы 

## Задание
Смоделировать задачу об обедающих философах:
- За круглым столом сидят 5 философов
- Чтобы поесть, каждому нужны две вилки — левая и правая
- Вилки лежат между философами, и ими можно пользоваться только по одной
- Философы чередуют состояния: думают → берут вилки → едят → кладут вилки → снова думают

## Как работает программа
- Каждый философ — это отдельный поток
- Вилки реализованы как массив мьютексов `forks[]`
- Чтобы поесть, философ блокирует две вилки:
  - Чётные философы берут сначала левую, потом правую
  - Нечётные философы — наоборот
- После еды философ освобождает вилки и снова начинает думать

## Особенность реализации
Такой протокол может привести к дедлоку
(ситуации, когда все философы ждут друг друга и никто не может поесть).  
Это сделано специально для показа проблемы синхронизации

## Файлы проекта
- `main.c` — создаёт потоки философов и инициализирует вилки
- `funcs.c` — содержит функцию `philosopher()`, которая описывает поведение философа
- `Makefile` — автоматическая сборка проекта

## Компиляция и запуск
Собрать программу:
```bash
make
```
Запустить:
```bash
./philo
```

Программа работает бесконечно. Чтобы остановить — нажмите Ctrl+C
