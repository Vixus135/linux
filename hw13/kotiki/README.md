# Упражнение 2: Покорми кота

## Задание
Сделать программу, где есть хозяин и 5 котов:
- Хозяин следит за кормушкой и подсыпает корм, если его меньше 100 г
- В кормушке не может быть больше 1000 г
- Каждый кот ест по 100 г за раз
- Коты могут есть одновременно

## Как работает программа
- В начале в кормушке 500 г корма
- Хозяин каждые 2 секунды проверяет миску. Если корма мало, он добавляет 500 г
- Коты по очереди берут по 100 г. Если корма меньше 100 г, они ждут, пока хозяин подсыпет
- Для синхронизации используется:
  - **Мьютекс (`mtx`)** — чтобы коты и хозяин не меняли количество корма одновременно
  - **Условная переменная (`cond`)** — чтобы коты могли ждать, пока хозяин подсыпет корм

## Почему программа работает правильно
1. Мьютекс защищает переменную `food` от одновременного доступа
2. Условная переменная позволяет котам "спать", пока корма недостаточно
3. Когда хозяин подсыпает корм, он будит всех котов (`pthread_cond_broadcast()`)
4. Коты проверяют условие в цикле `while`, чтобы избежать ложных пробуждений

## Файлы проекта
- `main.c` — создаёт потоки хозяина и котов
- `funcs.c` — содержит функции:
  - `owner()` — хозяин подсыпает корм
  - `cat()` — кот ест корм
- `Makefile` — автоматическая сборка проекта

## Компиляция и запуск
Собрать программу:
```bash
make
```
Запустить:
```bash
./cormim_kotikov
```
Программа работает бесконечно. Чтобы остановить — нажмите Ctrl+C
