## Содержание упражнений

### Упражнение 1: Генерация чисел и вычисления (`potoki`)
**Задача:** Реализовать приложение с тремя потоками:
- Поток 1 — каждую секунду генерирует случайное число.
- Поток 2 — если число чётное, печатает его квадрат.
- Поток 3 — если число нечётное, печатает его куб.

### Упражнение 2: Покорми кота (`kotiki`)
**Задача:** Смоделировать взаимодействие владельца и 5 котов:
- Владелец подсыпает корм в кормушку (100–1000 г).
- Каждый кот ест 100 г в день.
- Коты могут есть одновременно.
- Используются условные переменные и мьютексы для синхронизации.

### Упражнение 3: Обедающие философы (`philosofs`)
**Задача 1 (without):** Классическая реализация с возможным дедлоком.

**Задача 2 (with):** Реализация с атомарным захватом вилок:
- Используется `pthread_trylock`, мьютекс `forks` и условная переменная.
- Философы захватывают обе вилки или ждут, если одна недоступна.

### Упражнение 4: Спящий парикмахер (`parikmaher`)
**Задача:** Смоделировать работу парикмахерской:
- Один парикмахер, несколько клиентов.
- Парикмахер работает, если есть клиенты, иначе спит.
- Клиенты ждут в приёмной, если кресло занято.

## Требования для сборки и запуска
- Компилятор **gcc**
- Библиотека **pthreads**
- Операционная система **Linux/Unix**

Для просмотра каждого упражнения перейдите в соответствующую папку
и прочитайте файл README.md (в нём описана логика кода и инструкция по компиляции)
