# Упражнение 1: Генерация чисел и вычисления

## Задание
Нужно сделать программу с тремя потоками:
1. Первый поток каждую секунду создаёт случайное число
2. Второй поток проверяет: если число чётное — считает его квадрат
3. Третий поток проверяет: если число нечётное — считает его куб

## Как работает программа
- Все потоки используют одну общую переменную `number`
- Чтобы не было ошибок, доступ к этой переменной защищён мьютексом
- Поток-генератор каждую секунду создаёт число и сообщает другим потокам, что оно готово
- Потоки квадрат и куб ждут сигнала. Когда число появляется, они проверяют его и выводят результат

## Почему всё правильно
- Мьютекс не даёт потокам одновременно менять число
- Условная переменная позволяет потокам ждать новое число, не занимая процессор
- Когда генератор создаёт число, он будит оба потока
- Потоки проверяют чётность и выводят либо квадрат, либо куб

## Файлы проекта
- `main.c` — запускает три потока и управляет их работой.
- `funcs.c` — содержит функции для каждого потока:
  - `generator()` — создаёт числа.
  - `square()` — считает квадрат.
  - `cube()` — считает куб.
- `Makefile` — упрощает сборку программы.

## Сборка и запуск
Собрать программу:
```bash
make
```
Для запуска:
```bash
./potoci
```
Программа работает бесконечно. Чтобы остановить — нажмите Ctrl+C.
